# **App Name**: Elitefolio

## Core Features:

- Interactive 3D Element: Include a subtle 3D element (rotating cube or abstract shape) that subtly reacts to mouse movement.
- Dynamic Background Gradient: Implement a dark background with a subtle, shifting gradient that subtly animates and changes color over time.
- Project Filtering & Sorting: Allow users to filter projects by tags (e.g., 'Web Development', 'Mobile Apps', 'Data Science') and sort by date or relevance.
- Animated Transitions: Incorporate smooth transitions between sections/pages for a fluid user experience. Use subtle, modern animations for loading and interactions.
- AI-Powered Portfolio Review Tool: Users can submit their portfolio link and get instant feedback based on a database of successful portfolios and industry best practices. This tool can provide advice about the overall aesthetics, features and content.
- Contact Form with Validation: A simple contact form with email and validation for error handling

## Style Guidelines:

- Primary color: Subtle purple (#A090D3) for an elegant and high-tech vibe.
- Background color: Deep navy (#121A2B) for a modern, dark mode aesthetic.
- Accent color: Light blue (#79B4E2) to create subtle and elegant contrasts.
- Headline font: 'Space Grotesk' sans-serif for a techy feel, used for titles.
- Body font: 'Inter' sans-serif, modern, machined, objective, neutral, for body text.
- Minimal, geometric icons with a thin stroke to match the clean aesthetic.
- Use a clean, modern layout with ample whitespace. Prioritize content hierarchy with clear sections.
- Smooth, subtle animations for scrolling, hovering, and transitions.
